# Technical Analysis: NASA’s Mars Rover Autonomous Systems

## 📌 Project Overview
This repository contains a technical study of the engineering and robotic innovations behind NASA's Mars Rover missions. The analysis focuses on the transition from solar power to nuclear RTGs and the evolution of onboard autonomous navigation.

### 🤖 Core Robotics Insights
* **Autonomous Navigation:** Analysis of AI decision-making (AutoNav) used to mitigate Earth-Mars communication latency.
* **Environmental Engineering:** Study of material durability and thermal protection for -70°C Martian conditions.
* **System Redundancy:** Lessons learned from the failures of Spirit and Opportunity (e.g., dust accumulation on solar panels).
